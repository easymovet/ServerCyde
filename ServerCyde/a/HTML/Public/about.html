  • Speed and Scalability

What does this do for me?
  • You create "stored procedures" on our server that do something like connect to a DB or send an Email.
  • Then you put the script tag on your site it will give you a function to call those stored procs, that means you can connect to a DB or send and email without setting up a web server or giving up security.
  • As well as email and SimpleDB access it has user authentication calls with session management, comet messaging, and FBConnect.
  • Never thought you could make a secure site with purely client side code, well now you can.
